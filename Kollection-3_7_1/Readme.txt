Kollection is an expansion of MX Kollection Pro package, and includes the KTML 4, MX CSS Menus and XML Import-Export products. The new bundle helps web developers create dynamic websites quickly and easily by extending the capabilities of Dreamweaver.

The Kollection3_3_7_1.zip package contains
----------------------------------------------

- KTML_4/ - this folder includes version 4.1.6 of KTML 4, the online HTML editor that helps you edit and publish your website content directly in a browser
- MX_CSS_Menus/ - this folder includes version 2.1.1 of MX CSS Menus, a Dreamweaver extension that helps you create CSS navigation menus
- MX_Kollection3_Pro/ - this folder includes version 3.7.1 of MX Kollection 3 Professional, a Dreamweaver extension that allows users to easily create dynamic websites
- XML_Import-Export/ - this folder includes version 1.8.6 of XML Import-Export, a Dreamweaver extension that allows data exporting and importing
