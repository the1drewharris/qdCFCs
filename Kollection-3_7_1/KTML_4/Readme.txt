KTML4 is an Online HTML editor to be used as a replacement for a <textarea> in a form to allow non-technical users edit content visually.


The dwmx package contains
----------------------------------------------------------------------------------------

    docs/                               - this folder contains the KTML documentation
    KTML4Dev-4_1_6.mxp                  - the KTML4 extension, to install in the Macromedia Extension Manager
    KTML4-Help-4_1_6.mxp                - the KTML4 help extension
    License_KTML_Developer.rtf          - the KTML4 License file
    Readme.txt                          - this file
    The_Getting_Started_book.chm        - the Getting Started tutorial providing information about installing, uninstalling the product, defining a Dreamweaver site, defining a database connection.
