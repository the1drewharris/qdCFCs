The MX_CSS_Menus-2_1_1.zip package contains
----------------------------------------------

- docs
    - tutorials/ - this folder contains some site samples along with database dumps that you'll use when following the tutorials.
    - MX_CSS_Menus.chm - this file represents the MX CSS Menus documetation including the user manual and the tutorials
    
- MXCSSMenus2-2_1_1.mxp - the MX CSS Menus Dreamweaver extension.

- Getting Started.chm - the Getting Started tutorial providing information about installing, uninstalling the product, defining a Dreamweaver site, defining a database connection.

- skins
    - PSD files - the PSD files that will enable you to edit the images used by the CSS skins
    - PNG files - the PNG files that will enable you to edit the images used by the CSS skins
- README.txt file
- License.txt


MX CSS Menus installation procedure
------------------------------------------

1.Before installing our product please read the section allocated to this subject from the Getting Started.chm file, in order to avoid problems that might appear.

2.In order to uninstall the product please read the enclosed Getting Started.chm file.

3.If you are running our product on MAC, you can either install a special software for CHM support - http://chmox.sourceforge.net/ or you can read and browse the MX CSS Menus documentation from the InterAKT website.
